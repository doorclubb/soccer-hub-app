# Soccer Hub

A team management platform for soccer coaches and players.

## Features

- Live game tracking with playtime monitoring
- Roster management
- Goal scoring and statistics
- Formation/lineup builder
- Game history and results

## Deployment

This app is deployed on Vercel and uses Supabase for the database backend.

## Tech Stack

- HTML/CSS/JavaScript
- Supabase (Backend & Database)
- Vercel (Hosting)
